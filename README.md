# password-generator-python
Password Generator (Python)

This is a simple Python program that generates a strong random password.

Features

- User can choose the password length
- Uses letters, numbers, and special characters
- Generates a different password each time

How to run

1. Make sure Python is installed.
2. Download the repository.
3. Open a terminal in the project folder.
4. Run the program:

python password_generator.py

5. Enter the password length when prompted.

Example

Enter password length: 12
Generated password: A7!kP2#zQ9xL
